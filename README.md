## Features
<ul>
  <li><h3>Zero Data Collection Without users concern</h3></li>
  <li>
    <b>Adhan Timings:</b>
    <ul>
      <li>5 Prayer Times </li>
      <li>Sunrise Time</li>
      <li>Midnight Time (For Qiam-ul-layl)</li>
      <li>Last Third of night Time (For Qiam-ul-layl)</li>
    </ul>
  </li>
</ul>
